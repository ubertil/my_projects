Detailed Description

This class represents a GameRoom which role is to let 4 players play together. This let us abstract all the administration part of people joining and leaving a game. Once 4 players are connected to the same GameRoom, a game starts Ifany player leaves during the game, the session is restarted for everyone in the room.
